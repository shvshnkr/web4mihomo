Помощь: bootstrap mihomo и обновление с GitHub

В конфигурации mihomo один раз добавьте file proxy-providers с path, совпадающим с provider_yaml_path этого приложения, и две (или одну) группы с use: [<имя провайдера>].

{{ settings.bootstrap_yaml_hint() }}

Имя провайдера в mihomo должно совпадать с PROVIDER_NAME={{ settings.provider_name }}. External Controller: Bearer secret из MIHOMO_SECRET.

Обновление кода с GitHub: держите в репозитории только исходники; каталог data/ и файл .env добавьте в .gitignore на сервере (они уже в шаблоне проекта). Тогда git pull не перезапишет ваши прокси и секреты — они остаются на диске рядом с клоном.

Подробные логи в консоли uvicorn: по умолчанию включены (WEB4_VERBOSE_LOG=true). Краткий режим: WEB4_VERBOSE_LOG=false.
